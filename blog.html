My First Website

Date: 9 February 2026

Today I learned how to build my first multi-page website, and I have to say, it was both challenging and incredibly rewarding! I started with the basics - creating an index.html file and understanding the fundamental structure of HTML documents.

The journey began with learning about HTML tags like <header>, <nav>, <main>, and <footer>. I created three different pages: a home page, an about page, and a contact page. Linking them together using anchor tags felt like magic when I clicked between pages for the first time.

One of the biggest challenges was understanding the proper file structure and ensuring all my links worked correctly. I learned the importance of organizing files into folders and using relative paths. By the end of the day, I had a functioning multi-page website that I could navigate through smoothly.

The feeling of accomplishment when I opened my website in the browser and saw everything working perfectly was absolutely incredible. This is just the beginning of my web development journey!

#HTML #WebDevelopment #BeginnerCoder

Learning CSS Styling

Date: 10 February 2026

CSS helped me make my website look professional and attractive. After building the structure with HTML yesterday, I dove into the world of styling today, and wow - what a difference it makes!

I started by learning about CSS selectors and how to target specific elements on my pages. I experimented with colors, fonts, margins, and padding. The box model was confusing at first, but once I understood how content, padding, border, and margin work together, everything clicked.

I applied a color scheme to my website using complementary colors, added custom fonts from Google Fonts, and created a responsive navigation bar that looks great. I also learned about flexbox for layout, which made centering elements so much easier than I expected.

The transformation from my plain HTML website to a beautifully styled page was remarkable. I added hover effects to my buttons, created smooth transitions, and even implemented a simple animation. My website now looks like something a professional could have made, and I built it myself!

#CSS #Styling #WebDesign

JavaScript Basics and Interactivity

Date: 11 February 2026

Today marked an exciting milestone - I added my first JavaScript functionality to my website! After mastering HTML structure and CSS styling, it was time to bring my website to life with interactivity.

I started with the basics: variables, functions, and event listeners. My first achievement was creating a button that changes the background color of the page when clicked. It seems simple, but understanding how JavaScript interacts with the DOM (Document Object Model) opened up a whole new world of possibilities.

I then moved on to more complex features. I built a simple form validator that checks if the user has entered their name and email before submitting. I also created a dark mode toggle switch that saves the user's preference in localStorage, so their choice persists even after they close the browser.

The most exciting part was creating a simple image slider for my portfolio section. Watching images transition smoothly as users click the next and previous buttons made me realize the power of combining HTML, CSS, and JavaScript together. I'm learning that web development is not just about writing code - it's about creating experiences.

Next, I'm planning to explore JavaScript frameworks and learn about APIs to make my websites even more dynamic and data-driven!
